# Kubernetes Basics Workshop

Resources for a basic kubernetes workshop (e.g. k8s explained for podman/docker folks)

The [cluster-setup](cluster-setup) directory contains instructions to set up a local cluster with docker/podman

The [examples](examples) directory contains example kubernetes manifests applied during the workshop

The [slides](slides) directory contains the presentation slides

Client-side tools:

- kubectl
- [kubelogin](https://github.com/int128/kubelogin)
- [helm](https://github.com/helm/helm)
- [fluxcd](https://fluxcd.io/flux/installation/)
- [stern](https://github.com/stern/stern)
- [k9s](https://github.com/derailed/k9s)
- [kubectx](https://github.com/ahmetb/kubectx)
- jq/yq
